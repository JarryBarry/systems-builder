# 🚀 Enterprise Systems & Legacy Modernization Portfolio

## 🎯 The Mission
This repository documents my 48-week intensive engineering roadmap. My ultimate goal is to bridge the gap between legacy core enterprise systems (COBOL, Batch, DB2) and modern cloud infrastructure (Python, AWS, CI/CD, Docker). 

Instead of focusing solely on standard web development, this portfolio is geared toward **Enterprise Systems Engineering**—specifically modernizing, abstracting, and replacing legacy architectures using modern DevOps and CI/CD practices.
